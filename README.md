# YouTube Live Chat TTS (โปรแกรมอ่านแชทยูทูปอัตโนมัติ)

โปรแกรมนี้ออกแบบมาสำหรับสตรีมเมอร์ที่ต้องการให้มีเสียงอ่านแชทจากสตรีมสดบน YouTube แบบอัตโนมัติ โดยมีจุดเด่นคือรันในเครื่องตัวเอง (Local) กินสเปกน้อย มีการจัดการคิวที่ชาญฉลาด กรองคำหยาบ และมีลูกเล่นเสียงฮาๆ สำหรับการพิมพ์ข้อความรัวๆ

## ความสามารถหลัก
1. **Desktop App Mode:** เปิดขึ้นมาเป็นหน้าต่างโปรแกรมแยก ไม่ต้องเปิดแท็บในเว็บเบราว์เซอร์
2. **ระบบการเล่นเสียง (TTS):** ใช้เสียงที่สมจริงจาก Microsoft Edge TTS ไม่ต้องสมัคร API Key ใดๆ
3. **การจัดการคิวและสแปม (Anti-Spam & Flood Control):**
   - หากคนดูพิมพ์ข้อความเดียวกันรัวๆ ติดต่อกัน ระบบจะรวบอ่านแค่ข้อความเดียว
   - หากมีการพิมพ์ตัวอักษรยาวๆ (เช่น `5555555555`) ระบบจะย่อเป็นมุกฮาๆ เช่น `555 เลข 5 ล้านตัว` เพื่อไม่ให้อ่านลากเสียงยาวจนรบกวนสตรีม
4. **เลือกช่องเสียงออกได้ (Audio Output Device):** สามารถเลือกได้ว่าจะให้เสียงแชทไปออกที่หูฟัง ลำโพง หรือ Virtual Audio Cable สำหรับแยกเสียงใน OBS
5. **กรองคำหยาบ:** แปลงคำหยาบเป็นเสียงเซ็นเซอร์ (ตื๊ด)
6. **ทำงานเบื้องหลัง (Background Mode):** สามารถกดปิดกากบาทหน้าต่าง (X) เพื่อซ่อน UI ได้ โดยที่บอทจะยังคงอ่านแชทต่อไปเรื่อยๆ โดยไม่เกะกะหน้าจอ

## 🛠️ คู่มือการติดตั้งสำหรับผู้เริ่มต้น (Step-by-Step)

หากคุณเพิ่งเริ่มใช้งานและไม่มีความรู้เรื่องโปรแกรมมิ่ง ให้ทำตามขั้นตอนเหล่านี้:

### 1. ติดตั้งโปรแกรมพื้นฐาน
โปรแกรมนี้จำเป็นต้องใช้ **Python** ในการรัน
1. ไปที่เว็บไซต์: [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. กดปุ่มดาวน์โหลด Python เวอร์ชันล่าสุดสำหรับ Windows
3. เปิดไฟล์ติดตั้งที่โหลดมา **⚠️ สำคัญมาก:** ในหน้าจอแรกของการติดตั้ง ให้ติ๊กถูกที่ช่อง **"Add Python to PATH"** (หรือ Add python.exe to PATH) ด้านล่างสุดก่อนกด Install Now

### 2. ติดตั้งส่วนเสริมและการทำงานครั้งแรก
1. ไปที่โฟลเดอร์ของโปรแกรมนี้
2. ดับเบิ้ลคลิกที่ไฟล์ **`install.bat`** (ถ้ามี) เพื่อติดตั้งไลบรารีที่จำเป็น หรือเปิด Command Prompt ในโฟลเดอร์นี้แล้วพิมพ์:
   ```cmd
   python -m venv venv
   call venv\Scripts\activate
   pip install -r requirements.txt
   ```
   *(หมายเหตุ: หากไม่มีไฟล์ requirements.txt ให้รัน `pip install fastapi uvicorn edge-tts pytchat emoji pywebview pystray pillow`)*

### 3. วิธีการเปิดใช้งาน
1. เมื่อติดตั้งเสร็จแล้ว ในครั้งต่อๆ ไป ให้ดับเบิ้ลคลิกที่ไฟล์ **`run.bat`** เพื่อเปิดโปรแกรมได้เลย
2. รอสักครู่ จะมีหน้าต่างโปรแกรมเด้งขึ้นมา
3. ไปที่วิดีโอ Live สตรีมบน YouTube ก๊อปปี้ **Link URL** (หรือตัวอักษร Video ID หลัง v=)
4. นำมาวางในช่อง "ใส่ลิงก์ YouTube Live หรือ Video ID" แล้วกด **เริ่มต้น**
5. บอทจะเริ่มเชื่อมต่อและดึงแชทมาอ่านให้คุณฟังทันที!

### 4. การจัดการหน้าต่างโปรแกรม
- **อยากซ่อนหน้าต่าง:** สามารถกดปุ่มกากบาท (X) ที่มุมขวาบนของหน้าต่างได้เลย หน้าต่างจะหายไปแต่บอทยังคงอ่านแชทต่ออยู่เบื้องหลัง!
- **อยากเรียกหน้าต่างกลับมา:** ให้มองหาไอคอนคำว่า **"TTS"** ที่มุมขวาล่างของจอคอมพิวเตอร์ (System Tray บริเวณที่ดูนาฬิกา) คลิกขวาแล้วเลือก **"Show UI"**
- **วิธีปิดโปรแกรมถาวร:** คลิกขวาที่ไอคอน TTS ใน System Tray แล้วเลือก **"Quit"**

## เกร็ดความรู้ / การตั้งค่าเพิ่มเติม
- **การเปลี่ยนชื่อคนอ่านแบบรวดเร็ว (ด่วนสุด):** คุณสามารถ **คลิกขวาที่ชื่อ** ของคนในกล่องแชทได้เลย จะมีหน้าต่างเด้งขึ้นมาให้คุณตั้งชื่อเล่นได้ทันที!
- **เปลี่ยนชื่อคนอ่านแบบจัดการ:** สามารถคลิกตั้งค่า Alias (ชื่อเล่น) ด้านข้าง เพื่อเปลี่ยนชื่อช่องภาษาอังกฤษยาวๆ ให้บอทอ่านเป็นชื่อไทยสั้นๆ ได้
- **การจำค่าที่ตั้งไว้:** การตั้งค่าทั้งหมด (คำหยาบ ความยาวคำ ชื่อเล่น) จะถูกบันทึกไว้ในระบบ หากคุณเปิดโปรแกรมใหม่ ไม่ต้องตั้งค่าใหม่!
- **ปรับความดังเสียง:** สามารถปรับ Volume ได้ในหน้าต่างโปรแกรม
- **คำหยาบ:** สามารถเพิ่ม/ลด รายการคำหยาบที่ต้องการให้ระบบดูดเสียงได้ในการตั้งค่า

ขอให้สนุกกับการสตรีมมิ่งครับ! 🎉
